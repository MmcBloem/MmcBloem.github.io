Frequently Asked Questions

Q: How do I reset my game scores?

A: To reset your game scores, tap the "Reset Game Scores" button on the main screen. Confirm the reset when prompted. This will clear all current scores and round data.

Q: How can I start a new match?

A: You can start a new match by tapping the "Start New Match" button. This will reset the game state and begin a new session.

Q: Where is my game data stored?

A: All game data—including team names, scores, and round results—is stored locally on your device using Apple’s AppStorage. No data is sent to remote servers.

Troubleshooting

If you encounter any issues while using the app, please try the following steps:

  • Ensure your device is running the latest version of iOS.
  • Restart the app to clear any temporary glitches.
  • Reset the game scores if the app appears unresponsive.
  • Reinstall the app if problems persist.
